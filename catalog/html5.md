# HTML5

## Summary
HyperText Markup Language (HTML) merupakan suatu bahasa yang digunakan untuk pengembangan web, dimana pada saat ini versi dari dari HTML adalah 5.2 ([W3C](https://www.w3.org/)). Standar penulisan HTML5 adalah sebagai berikut :

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Sample page</title>
  </head>
  <body>
    <h1>Sample page</h1>
    <p>This is a <a href="demo.html">simple</a> sample.</p>
    <!-- this is a comment -->
  </body>
</html>
```
